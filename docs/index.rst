Welcome to Stockholm Commute's documentation!
=============================================

.. image:: https://team-eight.github.io/assets/img/logo.png
 :alt: Logo
 :align: center

.. centered::  II1305 Project | Team 8

Welcome to the Documentation belonging to Team 8's II1305 Project, STHLM Commute, for the version that ran in the spring of 2018.

The documentation is mainly for internal use of the team. It lists some basic descriptions regarding the structure of the project.
Team 8 is set up as an organization and all related projects to the team can be seen on `Github <https://github.com/Team-Eight>`_.

Are you a member?

- Take a look at the `FAQ - Adding to the Wiki <http://sthlm-commuter.readthedocs.io/FAQ.html#adding-to-the-wiki>`_ section, to see how you can contribute and/or make changes.

Project Website
^^^^^^^^^^^^^^^
The `project website <https://team-eight.github.io>`_, is hosted on GitHub, and can be found in a `separate repository <https://github.com/Team-Eight/team-eight.github.io>`_.
It gives a short description of the project and what it's about, some weekly updates & highlights of what we're doing.
